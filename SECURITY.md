# Security Policy

## Reporting
Open a GitHub Issue with the `security` label or email `sc@vspatabuga.io`. Sertakan detail endpoint/worker yang terpengaruh.

## Supported Versions
- Node.js 20 for dev/build.
- Wrangler CLI for deployment.

## Dependency Management
- Dependabot enabled for npm packages.
- Keep secrets out of repo (CF tokens, API keys).
